# GoalConnect Monorepo

Initial monorepo setup for the group project using npm workspaces.

## Structure

- `packages/ui-components`
- `packages/utils`
- `packages/feature-x`
- `packages/feature-y`
- `systems/` (individual assembly apps)

## Setup

1. `npm install`
2. `npm run build`
3. `npm run dev`
